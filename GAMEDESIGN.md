## 1. 게임 개요

### 한 줄 설명
묘지를 헤매는 영혼이 죽은 자들의 힘을 카드로 소환해 층층이 쌓인 저주를 돌파하는 모바일 덱빌딩 로그라이크.

### 핵심 재미
- 카드 조합으로 강력한 콤보를 터뜨리는 **버스트 전투**
- 런마다 달라지는 덱 구성 → **반복 플레이 동기**
- 다음 적 행동을 보고 판단하는 **전략적 선택**

### 레퍼런스
| 게임 | 참고 요소 |
|------|-----------|
| Slay the Spire | 전투 구조, 맵 구조, 덱빌딩 방식 |
| Inscryption | 묘지/죽음 세계관 |

---

## 2. 세계관

플레이어는 죽어서 묘지에 갇힌 **영혼(Soul)**이다.
저승으로 넘어가려면 묘지를 지키는 수문장들을 물리쳐야 한다.
묘지에 잠든 죽은 자들의 힘을 **카드**로 빌려 싸운다.

- **배경:** 어두운 묘지, 지하 납골당, 저승의 문
- **톤:** 다크하지만 가볍게 (픽셀아트)
- **주인공:** 이름 없는 영혼 (플레이어가 투영)

---

## 3. 핵심 게임플레이 루프

```
[런 시작]
    ↓
[맵 노드 선택] → 전투 / 이벤트 / 상점 / 보스
    ↓
[전투]
  플레이어 턴: 손패에서 카드 사용 (에너지 소비)
  적 턴: 예고된 행동 실행
  반복 → 승리 or 패배
    ↓
[보상] → 카드 추가 or 유물 획득
    ↓
[보스 처치] → 런 클리어
    ↓
[런 종료] → 다시 시작 (덱 초기화)
```

---

## 4. 전투 시스템

### 기본 자원
| 자원 | 설명 |
|------|------|
| **에너지** | 매 턴 3 회복. 카드 사용 시 소비 |
| **HP** | 런 내내 유지. 0이 되면 런 종료 |
| **방어도** | 매 턴 시작 시 0으로 초기화. 피해 먼저 흡수 |

### 턴 진행
```
[플레이어 턴 시작]
  - 에너지 3 회복
  - 카드 5장 드로우
  - 카드 사용 (에너지 소비)
  - "턴 종료" 버튼 클릭

[적 턴]
  - 예고된 행동 실행 (공격 / 방어 / 버프)
  - 다음 행동 예고 표시
```

### 상태이상 (공격 위주 3가지만)
| 상태이상 | 효과 |
|----------|------|
| **출혈(Bleed)** | 매 턴 종료 시 스택 수만큼 피해. 1씩 감소 |
| **약화(Weak)** | 공격력 25% 감소. 3턴 지속 |
| **취약(Vulnerable)** | 받는 피해 50% 증가. 2턴 지속 |

## 5. 카드 시스템

### 카드 구성 요소
```
[카드 이름]
비용: 에너지 소모량
효과: 실행 내용
타입: 공격 / 스킬 / 능력
희귀도: 일반 / 희귀 / 전설
```

### 초기 덱 (10장)
| 카드 | 비용 | 효과 |
|------|------|------|
| 영혼 강타 x5 | 1 | 적에게 6 피해 |
| 영혼 방어 x4 | 1 | 방어도 5 획득 |
| 공허의 일격 x1 | 2 | 적에게 15 피해 |

### 획득 가능 카드 (20장 목표)

**공격 카드 (12장)**
| 카드 | 비용 | 효과 |
|------|------|------|
| 저주의 낫 | 1 | 8 피해. 출혈 2 부여 |
| 영혼 폭발 | 2 | 18 피해 |
| 연속 베기 | 1 | 4 피해를 2회 |
| 취약 공격 | 1 | 6 피해. 취약 2턴 부여 |
| 저승 일격 | 3 | 30 피해 |
| 메아리 | 0 | 3 피해. 카드 1장 드로우 |
| 약화 일격 | 1 | 6 피해. 약화 2턴 부여 |
| 영혼 연쇄 | 2 | 5 피해를 3회 |
| 처형 | 2 | HP 30% 이하 적에게 즉사 |
| 피의 대가 | 1 | HP 4 소모. 20 피해 |
| 분노 | 1 | 이번 턴 공격력 +4. 드로우 1 |
| 망자의 절규 | 2 | 출혈 상태 적에게 출혈 스택×3 피해 |

**스킬 카드 (8장)**
| 카드 | 비용 | 효과 |
|------|------|------|
| 뼈 방패 | 1 | 방어도 8 획득 |
| 영혼 흡수 | 1 | 방어도 4. 카드 1장 드로우 |
| 고통의 의식 | 0 | HP 3 소모. 카드 2장 드로우 |
| 망각 | 0 | 손패 전체 버리기. 에너지 +2 |
| 죽음의 춤 | 2 | 방어도 15. 이번 턴 다음 공격 +8 |
| 공허 | 1 | 버리기 더미 카드 1장 재사용 |
| 저주 인장 | 1 | 적에게 취약+약화 각 2턴 부여 |
| 피의 계약 | 3 | 버리기 더미 전부 드로우 |

### 카드 희귀도 드롭률
| 희귀도 | 확률 |
|--------|------|
| 일반 | 60% |
| 희귀 | 35% |
| 전설 | 5% |

## 6. 적 시스템

### 행동 패턴 구조
적은 매 턴 시작 전 **다음 행동을 미리 표시**한다.
플레이어는 이를 보고 전략적으로 카드를 선택한다.

```csharp
// 패턴 예시
턴 1: 공격 8
턴 2: 방어 자세 (방어도 10)
턴 3: 강공격 14
→ 반복
```

### 일반 적 (5종)

**1. 떠도는 영혼**
- HP: 25 / 공격력: 낮음
- 패턴: 공격 5 → 공격 5 → 공격 8 (반복)
- 특징: 입문용, 위협 낮음

**2. 무덤 수호자**
- HP: 40 / 공격력: 중간
- 패턴: 방어 자세(방어도 8) → 공격 10 → 공격 10 (반복)
- 특징: 방어도를 먼저 깎아야 효율적

**3. 뼈 골렘**
- HP: 55 / 공격력: 높음
- 패턴: 공격 7 → 공격 7 → 강공격 18 (반복)
- 특징: 3번째 턴 강공격 대비 필요

**4. 저주받은 수도사**
- HP: 35 / 공격력: 중간
- 패턴: 약화 부여 → 공격 9 → 공격 9 → 강화(공격력 +3) (반복)
- 특징: 빨리 처리 안 하면 공격력 누적

**5. 묘지 망령**
- HP: 30 / 공격력: 중간
- 패턴: 공격 6 → 출혈 3 부여 → 공격 6 (반복)
- 특징: 출혈 누적 전에 빠르게 처리

### 보스 (2종)

**보스 1. 납골당의 수문장**
- HP: 120
- 패턴: 공격 10 → 방어도 15 → 공격 10 → 전체 공격 20 (반복)
- 특기: HP 50% 이하 시 공격력 +5

**보스 2. 저승의 문지기 (최종 보스)**
- HP: 180
- 패턴: 공격 12 → 취약 부여 → 강공격 22 → 재생(HP 15 회복) (반복)
- 특기: HP 60% 이하 시 패턴 속도 +1 (한 턴에 두 행동)

## 7. 맵 / 런 시스템

### 런 구조
```
[층 1] 노드 4개 → 보스 1
[층 2] 노드 5개 → 최종 보스
```

### 노드 종류
| 노드 | 비율 | 내용 |
|------|------|------|
| ⚔️ 전투 | 60% | 일반 적 1마리 전투 |
| ❓ 이벤트 | 20% | 선택지 이벤트 (HP↔카드 트레이드 등) |
| 🛒 상점 | 15% | 카드 구매 / 카드 제거 |
| 💀 보스 | 1개 | 층 마지막 보스 |

### 이벤트 예시 (5개)
1. **불길한 제단** — HP 10 소모 → 희귀 카드 1장 획득
2. **망자의 기억** — 카드 1장 제거 (덱 압축)
3. **저주의 샘** — 저주 카드 1장 추가 → HP 최대치 +15
4. **유령 상인** — 무작위 카드 3장 중 1장 무료 획득
5. **피의 제물** — 현재 HP 절반 → 전설 카드 1장 획득

### 상점
- 카드 3장 판매 (런당 랜덤)
- 카드 제거 서비스 (초기 약한 카드 정리용)
- 유물 1개 판매

### 보상 (전투 승리 후)
- 카드 3장 중 1장 선택 (또는 스킵)

## 8. 유물 시스템

런 중 획득하는 **패시브 강화 아이템** (5개 목표)

| 유물 | 효과 |
|------|------|
| 망자의 반지 | 매 전투 시작 시 방어도 4 획득 |
| 핏빛 검 | 공격 카드 피해 +2 |
| 영혼 결정 | 매 턴 시작 에너지 +1 (최초 1회만) |
| 저주의 두개골 | 적 처치 시 카드 1장 드로우 |
| 뼈 갑옷 | 방어도가 HP로 이월 (최대 10) |

## 9. 개발 일정 (3주)

### 1주차 — 코어 시스템
| 일차 | 작업 |
|------|------|
| 1~2일 | 카드 데이터 구조 (CardData, CardEffect) |
| 3~4일 | 덱 시스템 (드로우/핸드/버리기/셔플) |
| 5~7일 | 전투 상태머신 + 적 패턴 시스템 |

### 2주차 — 콘텐츠
| 일차 | 작업 |
|------|------|
| 1~2일 | 카드 20장 구현 |
| 3~4일 | 적 5종 + 보스 2종 구현 |
| 5~6일 | 맵 + 이벤트 + 상점 |
| 7일 | 유물 시스템 |

### 3주차 — UI / 폴리싱
| 일차 | 작업 |
|------|------|
| 1~2일 | 전투 UI (손패, HUD, 적 의도 표시) |
| 3~4일 | 맵 UI + 보상 UI |
| 5일 | 세이브/로드 |
| 6~7일 | 밸런싱 + 버그 수정 |
